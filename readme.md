# Proyecto Final - Tienda Online

Este proyecto es una tienda online básica desarrollada con HTML, CSS y JavaScript.

## Funcionalidades

- Visualización de productos desde una API REST.
- Carrito de compras dinámico con localStorage.
- Formulario de contacto funcional con Formspree.
- Diseño responsivo usando Flexbox y Grid.
- Buenas prácticas de accesibilidad y SEO básico.

## Tecnologías

- HTML5
- CSS3
- JavaScript (DOM, Fetch API, LocalStorage)
